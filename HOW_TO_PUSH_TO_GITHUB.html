⚡ STEP-BY-STEP GUIDE

Push to GitHub

Follow these steps to upload your Claude Trading Bot to GitHub so you can clone and run it on any machine, anytime.

📁 What You Have

📁 claude-trading-bot/
    📄 trading_bot.pymain bot
    📄 requirements.txtdependencies
    📄 run.shMac/Linux launcher
    📄 run.batWindows launcher
    📄 .env.exampleAPI key template
    📄 .gitignoreignores .env secrets
    📄 README.mddocumentation
⚠️ Your .env file (with your real API key) is in .gitignore — it will never be pushed to GitHub. Only .env.example (with a placeholder) gets pushed.
STEP 1

Create a GitHub Account (if you don't have one)

Go to github.com/signup and create a free account.

💡 Already have an account? Skip to Step 2.
STEP 2

Create a New Repository on GitHub

1. Go to github.com/new

2. Fill in the form:

Repository name: claude-trading-bot
Description: Claude-powered crypto paper trading bot with live terminal
Visibility: ⦿ Public (or Private — your choice)
Initialize: ✗ Do NOT check "Add a README" (you already have one)

3. Click the green "Create repository" button.

📋 Copy the URL of your new repo — it will look like:
https://github.com/YOUR_USERNAME/claude-trading-bot.git
STEP 3

Install Git (if not already installed)

Mac
Windows
Linux
$ # Check if Git is installed
$ git --version
$ # If not installed, install via Homebrew
$ brew install git

No Homebrew? Install it first: brew.sh

Configure Git (first time only)

$ git config --global user.name "Your Name"
$ git config --global user.email "your@email.com"
STEP 4

Move Your Files Into a Folder

Create a new folder on your computer and put all the bot files into it:

$ mkdir claude-trading-bot
$ cd claude-trading-bot
# Now copy all the downloaded files into this folder

Your folder should look like this:

📁 claude-trading-bot/
    📄 trading_bot.py
    📄 requirements.txt
    📄 run.sh
    📄 run.bat
    📄 .env.example
    📄 .gitignore
    📄 README.md
STEP 5

Initialize Git & Push to GitHub

Run these commands in order inside your claude-trading-bot folder:

① Initialize a git repository

$ git init

② Stage all files

$ git add .

③ Create your first commit

$ git commit -m "🚀 Initial commit — Claude trading bot"

④ Rename branch to main

$ git branch -M main

⑤ Connect to your GitHub repo

$ git remote add origin https://github.com/YOUR_USERNAME/claude-trading-bot.git
# ↑ Replace YOUR_USERNAME with your actual GitHub username

⑥ Push to GitHub

$ git push -u origin main
✅ Done! Your bot is now live at:
https://github.com/YOUR_USERNAME/claude-trading-bot
STEP 6

Run On Any Machine, Anytime

Whenever you want to use the bot on any computer:

① Clone the repo

$ git clone https://github.com/YOUR_USERNAME/claude-trading-bot.git
$ cd claude-trading-bot

② Set up your API key

$ cp .env.example .env
# Open .env and replace the placeholder with your real key:
# ANTHROPIC_API_KEY=sk-ant-your-real-key-here

③ Launch!

# Mac / Linux:
$ ./run.sh

# Windows:
$ run.bat
💡 The launcher automatically installs dependencies and checks your API key before starting.
STEP 7

Saving Future Changes to GitHub

If you modify the bot and want to save changes:

$ git add .
$ git commit -m "describe what you changed"
$ git push

To pull the latest version on another machine:

$ git pull

✅ Final Checklist

  • GitHub account created
  • New repository created at github.com/new
  • All bot files are in your local folder
  • Git initialized and files committed
  • Pushed to GitHub with git push -u origin main
  • Verified files appear on GitHub in browser
  • .env is NOT in GitHub (only .env.example)
  • Can clone and run with ./run.sh or run.bat
